## 🎪 Quick Start: Your First Joy Ride

```python
from story_weaver import JoyEngine, PartyStarter, HappinessGenerator

# Initialize maximum fun
engine = JoyEngine(happiness_level="INFINITE")

# Add your eccentric companions
engine.add_guest("Captain Funkatron")
engine.add_guest("Madame Plottwist")
engine.add_guest("DJ Narrative")

# Set the vibe
engine.set_soundtrack("upbeat_funk_mix")
engine.set_lighting("disco_ball_maximum")

# Write something that makes you SMILE
story = engine.create_story(
    theme="adventure_and_friendship",
    mood="euphoric",
    include=["music", "laughter", "surprise", "warmth"],
    exclude=["darkness", "sadness", "boredom"]  # Not welcome here!

# Export the joy
story.export(format="pure_happiness")
```
